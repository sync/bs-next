# bs-next

Minimal Next.js bindings written in ReasonML

## Installation

```shell
# yarn
yarn add @sync/bs-next

# or npm
npm install --save @sync/bs-next
```

Then add it to `bsconfig.json`:

```json
"bs-dependencies": [
  "bs-next"
]
```

## Publishing to NPM

```sh
yarn publish --minor --access public
```
